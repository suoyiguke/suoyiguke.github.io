RECORD LOCKS space id 1244 page no 3 n bits 96 index PRIMARY of table `test`.`test` trx id 346642 lock_mode X insert intention waiting
Record lock, heap no 1 PHYSICAL RECORD: n_fields 1; compact format; info bits 0

 0: len 8; hex 73757072656d756d; asc supremum;;

------------------

  • LOCK WAIT 2 lock struct(s), heap size 1136, 4 row lock(s)
    存在2个锁相关操作语句，4行记录被锁定

  • MySQL thread id 10, OS thread handle 18340, query id 247 localhost ::1 root update
    事务的线程ID信息，操作系统句柄信息，连接来源、用户

  • INSERT INTO test(id, a, b, c) VALUES (15, ‘new !!!!’, ‘bbb’, ‘ccc’)
    事务涉及的SQL

  • RECORD LOCKS space id 1244 page no 3 n bits 96 index PRIMARY of table test.test trx id 346642 lock_mode X insert intention waiting
    申请X锁，锁等待

  • Record lock, heap no 1 PHYSICAL RECORD: n_fields 1; compact format; info bits 0
